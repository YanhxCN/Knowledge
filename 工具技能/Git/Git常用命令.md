# Git基础

### Git工作模式

![](https://www.runoob.com/wp-content/uploads/2015/02/git-command.jpg)

- workspace：工作区

- staging area：暂存区

- local repository：本地仓库

- remote repository：远程仓库

  

- git config --global user.name

- git config --global user.email

- git init 初始化仓库

- git status

- git clone

- git log

- git add 添加到暂存区

- git commit

- git push

- git pull

- git reset

- git branch

- git checkout
